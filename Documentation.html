Valid auxiliary operands:
x(1,2,4,8...,32) - note length, the smaller the number the larger the length.
p - pause with length equal to the last length operand.
t(1 - 1000) - set tempo(values higher than 1000 are hardly audible).
s(1,2,4,8...,32) - time signature(base tone length, 1 beat = base tone length).

Before you start MPBeta if you wish the programme to work properly,
please create a file with an ".mp" extension containing at least one of the tone operands.
All auxiliary operands are not mandatory.
